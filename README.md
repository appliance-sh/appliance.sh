# appliance

Appliance is a platform to install and run applications on the cloud.

## Getting Started

1. Install the Appliance Command Line Interface (CLI)

```shell
npm install --global appliance.sh
```

2. Check that Appliance is correctly installed

```shell
appliance --version
```

3. Use Appliance to install and run applications on the cloud.

```shell
appliance install $application_name
```
